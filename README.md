# SQL- Rockbuster Stealth Video Rental Analysis

## Introduction:
Rockbuster Stealth LLC is a traditional movie rental company who is looking to break into the online rental/streaming space. Because they are facing stiff competition such as Netflix and Amazon Prime I will be utilizing skills as a data analysit to answer key questions in help with a launch strategy into this streaming platform and compete with these already established major companies using SQL.

## Key Questions:

    -What are the demographics of customers?
    -What countries are high value customer located?
    -Do sales vary pending geographic region?
    -WHat is the average rental period of movies?
    -What types of movies creat the most revenue?

## Tools
  When conducting analysis the primary tools used is pgAdmin4, Postgre SQL, and a relational database managemet system. By adding Rockbusters data set (film inventory, customers, deprtments, stores, staff, actors, transactions etc...) into the Postgre SQL database im then able to isolate and draw the necessary information to answer department specific questions using queries. These skills are transferrable across all departments and industries to better create actionable insights.
